## What is Chappie?

Chappie is not just a chat with AI. It's a full-featured agent that lives inside your project — reads files, writes code, makes commits, runs builds, and remembers everything across sessions.

You bring the task. Chappie brings the execution.

## Features

### 🤖 Any AI model, any provider
Connect OpenAI, Anthropic, Gemini, Ollama, or any OpenAI-compatible API.
217+ models available. Switch in one click.

### 💬 Smart chat
Streaming responses, Markdown rendering, syntax highlighting with Copy button, conversation history, file attachments.

### 🧠 3-Level Memory
Chappie remembers context across sessions, threads, and projects.
Global facts. Project decisions. Thread context. Nothing gets lost.

### 📁 Project mode
Connect a folder — Chappie becomes a full team member.
Read and write files, run git commands, build and test your project.

### 🛠 Developer tools
- **Git** — status, commit, revert, log, checkout — right from the chat
- **Build & test** — run Xcode, Swift Package Manager, Gradle, Cargo, and more
- **Localization** — manage `.xcstrings` with dedicated tools
- **Architecture files** — auto-maintained Mermaid diagrams
- **Web fetch** — read any URL, docs, or GitHub page without leaving the chat
- **Shell** — run any script directly from the conversation

### 🌍 10 interface languages
English, Русский, 中文, Español, العربية, Français, Deutsch, 日本語, Português, हिन्दी

### 🔄 Auto-updates
Chappie checks for updates automatically and installs them in the background via Sparkle.
